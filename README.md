# Freedium

Freedium is a python utility which let's you read membership articles for free.

## Installation

* Clone the repo 
* Install dependencies (preferably via [pipenv](https://pipenv.pypa.io/en/latest/))
  
## Usage

Run `main.py` and your result will be in `main.html`

    $ python main.py
